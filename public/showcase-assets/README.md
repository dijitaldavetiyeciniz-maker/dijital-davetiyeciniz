# Showcase Assets

This directory contains deterministic visual assets used solely for the Template Showcase Lab and end-to-end (E2E) testing environments.

These assets are AI-generated and royalty-free, intended to ensure a consistent and high-quality presentation when reviewing the Premium Templates. They simulate user-uploaded content in a "Photographed" state.

## Assets List
- `cinematic-couple.jpg`: Used for Cinematic Poster template testing.
- `polaroid-*.jpg`: Used for Polaroid Story collage templates.
- `night-sky.webp`: A wide starry background used for Night Sky templates.
- `parchment-texture.webp`: Used for Royal Letter / scroll layout testing.
- `hotel-lobby.jpg`: Used for the Luxury Hotel Invitation layout.
- `magazine-cover.jpg`: Used for the Fashion Magazine Editorial layout.

These images will not be used in live production for end-users unless a fallback is explicitly required.
